Custom Directives
ms-material-color-picker
Description:

ms-material-color-picker directive allows you to pick any of the Material Design colors.

Options:
Options Description
ms-model-type (string) Model type of the color picker. Can be either `class` or `obj`
ng-model (required) Value that will be updated on color pick
Example:

Depending on ms-model-type value, color picker will either update the model with the appropriate class name or the object.
